Во всех заданиях если количество возможных вариантов превышает 10^5 ограничтесь 10.000 экземплярами

для этого добавьте в код условие остановки по счётчику как ниже:
```python

  n = 0
  for ... :
    n = n + 1
    if n > 10000:
      break
      ...
```
